### All binaries using WLED Wemos shield pinout

- ESP8266 -> LED - GPIO2, GPIO0 - button(or CLK), Relay - GPIO12;
- ESP32 -> LED - GPIO16, GPIO17 - button(or CLK), Relay - GPIO19;
- (for other pins please see pinout table on schematic)
